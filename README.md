## Single-Cycle and Multi-Cycle Processor Design

## Overview

This repository contains an academic laboratory project focused on the design, implementation, and comparison of single-cycle and multi-cycle processor architectures.

The lab was completed as part of a Hardware Description Languages and FPGA-based digital design course. It focuses on processor microarchitecture, RTL implementation, datapath and control-unit separation, and functional verification.

The project demonstrates how different processor execution methodologies affect hardware complexity, timing, performance, and control logic design.

---

## Main Objectives

- Design and analyze a single-cycle processor core
- Design and analyze a multi-cycle processor core
- Understand the trade-offs between single-cycle and multi-cycle execution
- Separate the processor datapath from the control unit
- Implement processor logic using RTL methodology
- Verify the processor functionality using simulations and testbenches

---

## Key Topics

- Single-cycle processor architecture
- Multi-cycle processor architecture
